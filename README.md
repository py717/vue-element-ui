# waye-pro

> 通过 vue-cli 生成的项目脚手架

## 使用

```bash
git clone https://github.com/wayejs/waye-pro.git project-xxx
cd project-xxx
npm  i
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```
